# Exam Anxiety AI

This project reduces exam anxiety by generating daily micro-goals, tracking consistency, and showing a confidence score instead of ranks.

## Tech Stack
- **Backend:** FastAPI
- **Frontend:** Streamlit
- **Data:** Pandas / NumPy
- **Logic:** Rule + trend based AI

## Setup

1. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run Backend:**
   Navigate to the `backend` directory and run:
   ```bash
   cd backend
   uvicorn main:app --reload
   ```

3. **Run Frontend:**
   Navigate to the `frontend` directory and run:
   ```bash
   cd frontend
   streamlit run app.py
   ```
